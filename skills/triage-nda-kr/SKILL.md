# /legal-kr:triage-nda-kr -- 한국법 NDA 사전 심사

## Invocation

이 skill은 다음과 같은 상황에서 활성화됩니다:
- NDA, 비밀유지계약, 기밀유지약정의 검토를 요청할 때
- 비밀유지 조항의 한국법 적합성을 확인할 때
- `/legal-kr:triage-nda-kr` 슬래시 커맨드가 호출될 때

## Workflow

### Step 1: NDA 수신

사용자로부터 NDA를 수신하고 전체 구조를 파악합니다.

### Step 2: 컨텍스트 수집

- **당사자 위치**: 정보 제공자(Discloser) / 수령자(Recipient) / 상호(Mutual)
- **거래 맥락**: M&A 실사 / 기술 협력 / 고용 / 벤더 평가
- **국제 거래 여부**: 국내 / 국제 (준거법 확인 필요)

기본값: 상호 NDA, 수령자 측 검토

### Step 3: Playbook 로드

`legal.local.md`에서 비밀유지 관련 기준을 로드합니다.

### Step 4: 핵심 항목 분석

| 검토 항목 | 한국법 기준 |
|-----------|------------|
| **비밀정보 정의** | 범위가 명확한지, 과도하게 광범위하지 않은지 |
| **비밀유지 기간** | 한국 상관행: 1~3년 표준. 영업비밀은 부정경쟁방지법 기준 (비밀 유지 시 무기한 보호) |
| **허용 공개 범위** | 법적 공개 의무, 임직원/자문인 공개, 공지 정보 예외 |
| **경업금지 조항** | 한국법상 유효성: 기간·지역·업종 합리적 범위 내에서만 유효 (판례). 과도한 경업금지는 민법 제103조 위반 |
| **잔존 정보(Residuals)** | 수령자 기억에 남은 정보 사용 허용 여부 |
| **반환/파기 의무** | 계약 종료 시 비밀정보 반환 또는 파기 절차 |
| **구제 수단** | 금지청구(가처분), 손해배상, 위약금 수준 |
| **준거법/관할** | 국제 NDA 시 국제사법 검토, 한국법 관할 확보 |
| **개인정보 포함 여부** | 비밀정보에 개인정보 포함 시 개인정보 보호법 별도 적용 |

### Step 5: 위험도 분류

각 항목을 GREEN/YELLOW/RED로 분류합니다.

**한국법 특유 RED 기준**:
- 경업금지 기간 2년 초과 또는 지역·업종 무제한 → 무효 소지 (판례)
- 위약금이 실제 손해와 현저히 불균형 → 민법 제398조 감액
- 비밀유지 의무가 사실상 영업활동 전반을 제한 → 민법 제103조
- 일방적 NDA에서 수령자에게만 과도한 의무 → 약관규제법 적용 가능

### Step 6: 트리아지 결과 출력

## Output Format

```markdown
# NDA 트리아지 결과

## NDA 개요
- **유형**: 상호 NDA / 일방 NDA
- **당사자**: [Discloser] ↔ [Recipient]
- **거래 맥락**: [M&A 실사 / 기술 협력 등]

---

## 항목별 분석

- **의견**: [분석]

- **판정**: GREEN / YELLOW / RED
- **현재**: [N년 / 무기한]
- **의견**: [한국 상관행 대비 분석]

[... 각 항목 반복 ...]
```

## Notes

- NDA가 약관에 해당하면 (사업자가 다수와 체결하기 위해 일방적으로 작성) 약관규제법이 적용됩니다.
- 영업비밀 보호는 부정경쟁방지법 제2조 제2호에 의해 NDA 없이도 보호되나, NDA는 보호 범위와 입증을 용이하게 합니다.
- 국제 NDA의 경우 국제사법에 따른 준거법 결정이 중요합니다. 한국법이 강행적으로 적용되는 영역(개인정보 보호법 등)을 별도로 안내합니다.
