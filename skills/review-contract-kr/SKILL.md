# /legal-kr:review-contract-kr -- 한국법 계약서 검토

## Invocation

이 skill은 다음과 같은 상황에서 활성화됩니다:
- 사용자가 계약서, 이용약관, 서비스 계약 등의 검토를 요청할 때
- 한국법 기준의 법적 위험도 분석이 필요할 때
- `/legal-kr:review-contract-kr` 슬래시 커맨드가 호출될 때

## Workflow

### Step 1: 계약서 수신

사용자로부터 계약서를 수신합니다:
- 텍스트 직접 입력
- 파일 첨부 (PDF, DOCX, TXT)
- URL 링크

계약서를 수신하면 먼저 전체를 읽고 조항 구조를 파악합니다.

### Step 2: 컨텍스트 수집

다음 정보를 사용자에게 확인합니다 (제공되지 않은 경우):

- **당사자 위치**: 갑(서비스 제공자) / 을(고객) / 중립
- **계약 유형**: B2B / B2C(소비자) / 고용 / 라이선스
- **관심 영역**: 특별히 주의할 조항이 있는지 (예: 손해배상, 해지, IP)
- **마감**: 검토 시급성

제공되지 않으면 기본값 적용:
- 당사자 위치: 을(고객) 측
- 계약 유형: 맥락에서 추론
- 관심 영역: 전체 검토

### Step 3: Playbook 로드

`legal.local.md` 파일을 찾아 조직의 계약 검토 기본 포지션을 로드합니다.

- Playbook이 있으면 → 해당 기준으로 판정
- Playbook이 없으면 → "No playbook found. Using general Korean law standards." 안내 후 일반 한국법 기준 적용

### Step 4: 조항별 분석

계약서를 조항 단위로 분할하여 각 조항을 분석합니다.

**분석 카테고리** (한국 계약 관행 기준):

| 카테고리 | 핵심 검토 법령 |
|----------|---------------|
| 손해배상 제한 | 약관규제법 제7조, 제8조, 민법 제393조, 제398조 |
| 계약 해제/해지 | 약관규제법 제9조, 민법 제543조~제553조 |
| 면책/보증 | 약관규제법 제7조, 민법 제580조 |
| 비밀유지 | 부정경쟁방지법, 약관규제법 제12조 |
| 지식재산권 | 저작권법 제9조, 발명진흥법 (직무발명) |
| 개인정보 보호 | 개인정보 보호법 제26조, 제28조의8 |
| 준거법/관할 | 약관규제법 제14조, 국제사법 |
| 대금 지급 | 하도급법 제13조, 상법 (상사법정이율) |
| 경업금지/전속 | 민법 제103조, 판례 기준 합리성 |
| 자동갱신/통지 | 약관규제법 제12조, 전자상거래법 |

### Step 5: 이탈 사항 분류

각 조항을 Playbook 기준으로 분류합니다:

- **GREEN**: Playbook 수용 범위 내. 한국법상 유효성에 의문 없음.
- **YELLOW**: Playbook 기본 포지션에서 이탈. 협상 또는 수정 여지 있음. 해석에 따라 결론 달라질 수 있음.
- **RED**: 한국법 강행규정 위반 소지 또는 Playbook 에스컬레이션 기준 해당. 수석 변호사/외부 자문 검토 필요.

### Step 6: Redline 제안

YELLOW/RED 조항에 대해 수정 문안을 제안합니다:

```
**현재 문안**: "을의 손해배상 책임은 직접 손해에 한하며..."
**수정 제안**: "을의 손해배상 책임은 직접 손해에 한하되, 배상 상한은 본 계약 연간 이용료의 100%로 한다. 단, 고의 또는 중대한 과실로 인한 손해는 본 제한에서 제외한다."
**수정 근거**: 약관규제법 제7조 제1호 — 고의·중과실 면책 금지. 배상 상한 명시로 분쟁 리스크 감소.
```

### Step 7: 비즈니스 임팩트 요약

법률 분석 결과를 비즈니스 관점에서 요약합니다:
- 주요 재무적 리스크 (예: 무제한 배상 노출)
- 운영 리스크 (예: 일방적 해지 → 서비스 연속성)
- 협상 우선순위 제안

### Step 8: 심층 분석 안내

YELLOW/RED 조항에 대해 adversarial 심층 분석을 안내합니다:

> 이 조항에 대해 3-agent 교차 검증 심층 분석이 필요하시면
> `/legal-kr:adversarial-review <해당 조항>` 을 실행하세요.

## Notes

- 이 skill은 법률 자문이 아닌 법률 워크플로우 보조 도구입니다. 모든 분석은 자격을 갖춘 법률 전문가의 검토를 거쳐야 합니다.
- 약관규제법은 사업자가 다수 고객과 체결하기 위해 작성한 약관에 적용됩니다. 개별 교섭이 이루어진 계약 조항에는 적용되지 않을 수 있습니다.
- B2B 계약이라도 약관에 해당하면 약관규제법이 적용됩니다 (대법원 판례).
- MCP tool (~~korean-law)이 연결된 경우, 관련 법령 조문과 판례를 실시간 검색하여 인용합니다. 연결되지 않은 경우 일반 법률 지식을 사용합니다.
